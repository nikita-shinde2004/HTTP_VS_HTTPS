What's Happening?

In the HTTP login, your username and password are sent as plain text. Any attacker using tools like Wireshark on the same network can see exactly what you typed.

In the HTTPS login, your credentials are encrypted with TLS before being sent. Even if intercepted, the attacker only sees random encrypted bytes — completely useless without the decryption key.
